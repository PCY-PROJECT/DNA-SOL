# 复盘流程参考

## 日终复盘结构

```
1. 今日交易摘要
   - 交易次数
   - 盈亏汇总（来自真实账户数据）
   - 胜率（仅统计，不承诺未来表现）

2. 每笔交易回顾
   - 计划 vs 实际执行
   - 触发条件是否成立
   - 离场时机判断

3. 决策质量评估
   - 风控规则是否遵守
   - 仓位是否合理
   - 止损是否执行

4. 改进点
   - 下次可以改进的执行细节
   - 数据/信息缺口

5. 下一交易日准备
   - 需要关注的关键价位
   - 待观察的市场条件
```

## 要求

- 所有盈亏数据来自 account-read MCP 真实结果
- 不生成假盈利数据
- 不对未来交易做盈利承诺
